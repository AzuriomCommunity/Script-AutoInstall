## Usage
```sh
wget https://raw.githubusercontent.com/MaximeMichaud/Azuriom-install/master/azuriom-install.sh
chmod +x azuriom-install.sh
./azuriom-install.sh
```
## Compatibility
Please consider latest versions as more stable.

| Debian 8 |❔|
